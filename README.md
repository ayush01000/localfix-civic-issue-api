\# LocalFix - Civic Issue Reporting API



LocalFix is a Django REST Framework backend project for reporting and managing civic issues such as potholes, garbage, water leakage, broken streetlights, and other local problems.



\## Tech Stack



\- Python

\- Django

\- Django REST Framework

\- SQLite for development

\- Git and GitHub



\## Features Planned



\- User registration and login

\- Citizen, Staff, and Admin roles

\- Issue reporting API

\- Image uploads

\- Issue assignment

\- Status tracking

\- Notifications

\- Admin analytics dashboard

\- Filtering and search

\- API documentation
